Telegram-бот на Kotlin для проверки суммарного баланса криптокошельков в USD. Бот позволяет пользователю добавлять адреса кошельков, хранить персональные профили и получать оценку баланса по нескольким блокчейнам и токенам.

Проект ориентирован на мониторинг мультичейн-кошельков и поддерживает EVM-сети, Solana и TON.

## Возможности

- Проверка баланса криптокошельков через Telegram.
- Поддержка нескольких профилей пользователя.
- Хранение адресов кошельков по категориям:
  - `eth`
  - `sol`
  - `ton`
- Расчет общей стоимости активов в USD.
- Поддержка EVM-сетей:
  - Ethereum Mainnet
  - Base
  - Arbitrum
  - Polygon
  - Optimism
  - Abstract
  - BSC
  - ZkSync
- Поддержка Solana SPL-токенов.
- Поддержка TON и Jetton-токенов.
- Кэширование цен токенов.
- Получение цен через внешние API.
- Персональная конфигурация пользователя.
- Настройки через inline-кнопки Telegram.
- Поддержка локализации:
  - English
  - Русский
- Хранение данных в PostgreSQL.
- Миграции базы данных через Liquibase.

в основном в проекте заложен какой-никакой фундамент, но требуются доделки
